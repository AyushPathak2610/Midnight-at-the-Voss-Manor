## Tips

### Voice Generation
1. **Copy text exactly** (including punctuation and ellipsis)
2. **Set stability values** as specified (important for character consistency)
3. **Enable Speaker Boost** for all voices
4. **Download as MP3** format
5. **Name files exactly** as shown (case-sensitive)

### Music Generation
1. Music is **optional** - game works great without it
2. If Sound Generation isn't available, skip music files
3. You can use any 60-second atmospheric music instead
4. Just name them correctly (intro.mp3, foyer.mp3, etc.)

### File Naming
- **Must be exact** - system looks for specific filenames
- **Use lowercase** for all filenames
- **Use underscores** not spaces
- **Extension must be .mp3**
